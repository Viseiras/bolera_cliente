# Proyecto Bolera

## Introducción
Este proyecto es una aplicación web desarrollada con React que simula la gestión de una bolera. Permite a los usuarios gestionar jugadores, realizar reservas de pistas y disfrutar de un minijuego de bolos animado. El objetivo es ofrecer una experiencia interactiva y sencilla para la administración y el entretenimiento en una bolera.

## ¿Para qué sirve este proyecto?
El proyecto está diseñado para facilitar la gestión de una bolera, permitiendo:
- Registrar y listar jugadores.
- Reservar horarios para jugar.
- Visualizar y administrar las reservas existentes.
- Jugar a un minijuego de bolos animado integrado en la aplicación.

## ¿Cómo usar el proyecto?
1. Abre el proyecto en Visual Studio Code.
2. Ejecuta el siguiente comando para instalar las dependencias necesarias:
   
	npm install react-router-dom

3. Inicia el servidor de desarrollo con:
   
	npm run dev

4. Accede a la aplicación desde tu navegador en la URL que se muestra en la terminal.
